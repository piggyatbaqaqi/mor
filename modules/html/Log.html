NAME

mor::Log - Log handler

SYNOPSIS

        use mor::Utilities;
        use mor::Log;
        my $log = mor::Log->new("log/file/name");
        $log->print("any message you want to record");

DESCRIPTION

A very simple object that takes a file name, in the form of a string, as its only parameter. It uses the getUniqueFileName found in mor::Utilites to always create a new log.

Methods
