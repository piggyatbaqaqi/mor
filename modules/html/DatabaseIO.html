NAME

mor::DatabaseIO - An object that can be used to access a database.

SYNOPSIS

        use mor::Settings;
        use mor::Utilities;
        use mor::Log;
        use Bio::TreeIO;
        use Bio::AlignIO;
        use Bio::SeqIO;

        my $DBIO = mor::DatabaseIO->new($databaseName, $userName, $password);
        $DBIO->connect();
        my $results = $DBIO->query($MySQLquery);

DESCRIPTION

Precondition
 This module requires the settings hash and the log file to exist and be
 initialized.

This is pretty much a wrapper for DBI. It saves lots of code in other modules and allows for more specialized code relating to mor.
